# Atendimento-automatico-WhatsApp

## Descrição:

O projeto consiste em uma aplicação de Inteligencia artificial, no caso utilizado o ChatGPT 4-o, para ser um assistente virtual para WhatsApp com contexto

Porque foi criada essa aplicação:

Para melhora no atendimento no WhatsApp, com um assistente 24h disponivel que responde de forma automatiza as menssagens dos usuarios.

## como foi construido:

foi utilizada as seguintes tecnologias:

-evolution-api (Api não oficial do WhatsApp)
-Fasapi  
-Agno  
-Docker-compose
